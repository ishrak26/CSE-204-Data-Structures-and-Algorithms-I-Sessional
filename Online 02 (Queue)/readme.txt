Source files included in this directory:
1. Stack.h: An abstract class defining what functions any implementation of a Stack ADT has to implement.
2. Arr.cpp: An array-based implementation of the Stack ADT.
3. LL.cpp: A Linked-list based implementation of the Stack ADT.
4. main.cpp: A main function to test the array based and linked list based implementation of the Stack ADT.
5. dw.cpp: An implementation of the given dishwasher simulation using any of the array based, linked-list based or One Array Two Stack based implementation of Stack ADT.
6. Arr1.cpp: An array-based implementation of the Stack ADT including insert and remove functions.
7. LL1.cpp: A Linked-list based implementation of the Stack ADT including insert and remove functions.
